# Backoffice Inventory UI

This directory contains a minimal Next.js workspace for the backoffice inventory pages.

## Run locally

```bash
cd services/api/uis/backoffice
npm install
npm run dev
```

Set `NEXT_PUBLIC_INVENTORY_API_URL` if the API is not running on `http://localhost:8000`.

Inventory pages are served under `/backoffice/inventory`.
